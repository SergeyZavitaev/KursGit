# Инструкция по работе с Git

Первым делом нужно установить Git на ваш компьютер и после чего начинать изучение

### Глава 1: Создание проекта

1. Для начала нужно прописать команду *git init* 
для создания репозитория
2. После чего можно добавить файлы проекта
командой: *git add название файла с расширением*

Привер: *git add example.txt*

3. После чего можно написать *git status* что бы узнать что ещё осталось добавить в файлы

 ### Глава 2: Создание commit и переключение между ними

 1. Что бы сделать сохранение нужно сделать commit - сохранение, фиксирование с помощью команды:
 *git commit -m "комментарий в кавычках"*

 (Рекомендуется делать commit после например создания нового функционала, исправления множества ошибок или завершения рабочего дня)

  2. Что бы переключаться между версиями commit нужно сначала увидеть список всех commit по команде;
 git log которая отобразит все commit где будут написаны id(пример e5frg45657 и т.д.) всех сохранений (что бы выйти из режима наджмите клавишу q с включённой английской раскладкой)
 3. После того как мы нашли интересующее нас сохранение, пишем команду 
 git checkout первые 4 символа сохранения
 4. Что бы вернуться к нынешнему сохранению надо написать
 *git checkout master*

  ### Глава 3: Дополнительные функции и сохранение изменений

  1. Что бы сохранить изменения в файле можно прописать команду
  *git add название файла с расширением* или сделать commit с сохранением всех файлов командой
  *git commit -am "название сохранения в кавычках"*
  2. Что бы посмотреть разницу между коммитами можно прописать команду
  *git diff первые 4 символа коммита*
